# Michael Jr — Personal Portfolio 

A personal profile page built as part of my Week 1 HTML practice — my first step toward becoming a software engineer.

## About
This is my developer profile page, showcasing who I am, my background, my skills, and where I'm headed. I'm a Computer Science student at Mulungushi University based in Lusaka, Zambia, on a mission to become a world-class software engineer.

## Sections
- **Introduction** — quick intro and call-to-action buttons
- **About Me** — my story and what drives me
- **Skills** — technical and soft skills
- **Projects** — showcase of my practice projects (ZedFresh, Task Manager, ZedWeather)
- **Education** — my academic background in table format
- **Goals** — where I'm headed in my career
- **Contact** — a working contact form plus my social links

## Built With
- HTML5 (semantic markup)

## What I Practiced
- Semantic tags: `<header>`, `<nav>`, `<section>`, `<article>`, `<address>`
- HTML tables with `<thead>`/`<tbody>`
- Ordered and unordered lists
- Forms with proper labels and input types
- Linking to external profiles (GitHub, LinkedIn)
